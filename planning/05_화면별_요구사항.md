# 화면별 요구사항

## 1. 공통 화면 요구사항

### 1.1 헤더

- 로고와 기관명 표시
- 6개 상위 메뉴와 2단 메뉴
- 통합 검색 버튼
- 모바일 햄버거 메뉴
- 현재 메뉴 위치 표시
- 스크롤 시 축소형 고정 헤더 적용 가능

### 1.2 푸터

- 기관 기본정보를 데이터베이스 또는 환경설정에서 공통 관리
- 개인정보 처리방침 강조
- 대표 연락처, 이메일, 주소
- SNS와 관련 사이트 링크
- 저작권 표기

### 1.3 공통 게시판

- 제목, 분류, 등록일, 조회수
- 검색: 제목+내용, 제목, 태그
- 필터: 분야, 대상, 기관, 연도
- 페이지네이션
- 첨부파일
- 이전글·다음글
- 공유와 인쇄
- 관련 콘텐츠 추천

### 1.4 공통 상태 화면

- 검색 결과 없음
- 등록된 게시물 없음
- 접근 권한 없음
- 페이지를 찾을 수 없음
- 서버 오류
- 로딩 중

## 2. 메인 페이지 `/`

### 2.1 목적

기관의 정체성, 핵심사업, 최신 정보, 참여 경로를 한 화면에서 전달한다.

### 2.2 화면 구성

① 메인 비주얼  
- 핵심 문구
- 보조 설명
- `주요사업 보기`, `협력 문의` 버튼
- 영상보다 최적화된 이미지 또는 가벼운 애니메이션 우선

② 핵심사업  
- 6개 사업 카드
- 아이콘, 사업명, 한 줄 설명, 상세 링크

③ 교육·행사  
- 모집중·예정·마감 상태
- 일정, 대상, 장소
- 신청 버튼

④ 최신 정보  
- 공지사항
- 정책·산업정보
- 자료실

⑤ 수행사례·성과  
- 대표 사례 3건
- 검증된 실적 수치

⑥ 협력기관  
- 로고 슬라이더 또는 그리드
- 공개 동의된 기관만 표시

⑦ 참여 유도  
- 사업 협력 제안
- 기업 지원 문의
- 전문가 등록

### 2.3 관리자 기능

- 섹션 노출 여부와 순서
- 메인 배너 등록
- 대표 게시물 선택
- 성과 수치 수정
- 협력기관 노출 순서 설정

## 3. 기관개요 `/about/overview`

- 설립 배경
- 설립 목적
- 주요 역할
- 법적 형태와 등록정보
- 기관 소개서 다운로드
- 허위 또는 확정되지 않은 정보는 게시하지 않음

## 4. 인사말 `/about/greeting`

- 대표자 사진
- 핵심 메시지
- 인사말 본문
- 대표자 직함과 성명
- 이미지 대체 텍스트 제공

## 5. 비전·핵심가치 `/about/vision`

- 비전 문장
- 미션
- 핵심가치 3~5개
- 전략목표
- 사업과 전략목표 연결도

## 6. 연혁 `/about/history`

- 연도별 타임라인
- 주요 설립, 협약, 사업, 발간, 수상 기록
- 연도 필터
- 증빙 가능한 내용만 등록

## 7. 조직도 `/about/organization`

- 조직도 이미지와 텍스트 구조 병행
- 부서별 역할
- 담당업무와 대표 연락처
- 개인 휴대전화와 불필요한 개인정보는 미노출

## 8. 오시는길 `/about/location`

- 주소, 교통편, 주차 정보
- 지도 API 또는 외부 지도 링크
- 모바일 지도 앱 연결
- 본문과 푸터의 주소·전화가 동일한 공통 데이터 사용

## 9. 주요사업 목록 `/business`

- 사업 분야별 카드
- 대상, 지원 내용, 진행 형태
- 관련 수행사례와 교육과정 연결
- 분야 필터

## 10. 주요사업 상세 `/business/:slug`

- 사업명과 핵심 요약
- 추진 배경
- 대상
- 지원·수행 내용
- 진행 절차
- 기대 효과
- 관련 사례
- 담당 부서
- 문의 버튼

## 11. 수행사례 목록 `/cases`

- 카드형 목록
- 분야, 연도, 대상기관, 지역 필터
- 대표 이미지
- 주요 성과 수치

## 12. 수행사례 상세 `/cases/:slug`

- 과제 또는 사업 개요
- 고객·협력기관
- 문제 상황
- 수행 내용
- 적용 기술·방법
- 성과
- 사진·산출물
- 관련 사업
- 공개 범위와 저작권 확인

## 13. 교육·행사 목록 `/programs`

- 탭: 전체, 교육, 세미나, 포럼, 행사
- 상태: 모집중, 예정, 마감, 종료
- 대상, 지역, 온·오프라인 필터
- 일정순 정렬
- 달력 보기와 목록 보기

## 14. 교육·행사 상세 `/programs/:id`

- 제목과 상태
- 과정 소개
- 대상과 선수지식
- 교육 목표
- 커리큘럼
- 강사
- 일정·장소·비용
- 모집인원과 신청기간
- 준비물
- 문의처
- 신청 버튼
- 첨부파일

## 15. 참가 신청 `/programs/:id/apply`

### 15.1 입력 항목

- 이름
- 연락처
- 이메일
- 소속
- 직무 또는 관심 분야
- 신청 동기
- 개인정보 수집·이용 동의
- 선택적 마케팅 동의는 별도 체크

### 15.2 처리 요구

- 필수값 검증
- 중복 신청 확인
- 접수 완료 번호
- 담당자 이메일 알림
- 신청자 확인 이메일
- 관리자 상태: 접수, 검토, 승인, 대기, 취소

## 16. 정책·산업정보 `/insights`

- 분야: AI, 소프트웨어, 스마트산업, 교육, 보안, 지역혁신 등
- 출처 기관
- 발행일
- 핵심 요약
- 원문 링크
- 관련 사업·교육 연결
- 외부자료임을 명확히 표시

## 17. 정부지원사업 `/opportunities`

- 지원기관
- 지원 대상
- 지원 내용
- 신청 기간과 마감일
- 지역
- 원문 공고 링크
- 마감·진행중 상태 자동 표시
- 공고 원문을 무단 복제하지 않고 요약과 링크 중심으로 운영

## 18. 자료실 `/resources`

- 분류: 연구보고서, 교육자료, 발간물, 서식, 기타
- 파일 형식과 용량 표시
- 미리보기 가능 여부
- 다운로드 수
- 저작권·이용 조건

## 19. 공지사항 `/notices`

- 중요 공지 상단 고정
- 일반 공지 목록
- 게시 기간 설정
- 첨부파일
- 예약 게시

## 20. 활동소식 `/news`

- 카드형 목록
- 사진, 날짜, 요약
- 협약, 교육, 행사, 언론, 프로젝트 분류
- 갤러리 포함 가능

## 21. 협력기관 `/partners`

- 기관 로고와 기관명
- 협력 분야
- 협약일 또는 협력 내용
- 외부 사이트 링크
- 협력 종료 또는 비공개 처리 가능

## 22. 문의·제안 `/contact`

### 22.1 문의 유형

- 교육 문의
- 기업 지원
- 공공사업 협력
- 산학연 협력
- 강의·자문
- 언론·홍보
- 기타

### 22.2 입력 항목

- 기관·소속
- 담당자명
- 이메일
- 연락처
- 문의 유형
- 제목
- 내용
- 첨부파일
- 개인정보 동의

### 22.3 처리 기능

- 접수번호 발급
- 스팸 방지
- 관리자 담당자 배정
- 상태: 접수, 확인중, 답변완료, 보류
- 처리 이력 기록

## 23. 통합 검색 `/search`

- 검색 대상: 사업, 교육, 공지, 소식, 정책정보, 자료실
- 결과별 콘텐츠 유형 표시
- 제목과 본문 검색어 강조
- 분류·기간 필터
- 검색 결과가 없을 때 추천 검색어 제공

## 24. 관리자 화면

### 24.1 로그인 `/admin/login`

- 이메일 기반 로그인
- Firebase Auth 또는 동등한 인증 적용
- 비밀번호 재설정
- 반복 실패 제한
- 관리자 계정만 접근

### 24.2 대시보드 `/admin`

- 게시물 수
- 모집중 교육
- 신규 신청
- 미처리 문의
- 최근 활동 로그

### 24.3 콘텐츠 편집기

- 제목, 요약, 본문, 분류, 태그
- 대표 이미지와 첨부파일
- 공개·비공개·예약
- SEO 제목과 설명
- 미리보기
- 변경 이력

## 25. 비기능 요구사항

| 구분 | 요구사항 |
|---|---|
| 반응형 | 360px 이상 모바일부터 데스크톱까지 대응 |
| 접근성 | 대체 텍스트, 키보드 탐색, 명도 대비, 폼 라벨 |
| 성능 | 이미지 WebP/AVIF, 지연 로딩, 코드 분할, 캐시 |
| SEO | 고유 title·description, Open Graph, sitemap.xml, robots.txt |
| 보안 | HTTPS, 입력 검증, 파일 확장자 제한, 인증·권한, 요청 제한 |
| 개인정보 | 최소 수집, 동의 기록, 보유기간, 삭제 절차 |
| 백업 | DB 자동 백업과 복구 절차 |
| 로그 | 관리자 작업, 오류, 보안 이벤트 기록 |
