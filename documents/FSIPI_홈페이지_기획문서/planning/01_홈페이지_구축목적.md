# 미래스마트산업진흥원 홈페이지 구축 목적

- 기관명: 미래스마트산업진흥원
- 영문명: Future Smart Industry Promotion Institute
- 약칭: FSIPI
- 문서 작성일: 2026년 6월 20일
- 구축 방식: 바이브코딩 기반 단계적 개발
- 참고 자료: `svakorea.org.zip`에 포함된 한국스마트빌리지협회 공개 웹페이지

## 1. 구축 배경

미래스마트산업진흥원은 AI, 디지털 전환, 스마트산업, 교육, 기업 지원, 산학연 협력과 관련된 사업을 소개하고 이해관계자와 연결할 공식 온라인 창구가 필요하다. 초기 홈페이지는 단순 기관 소개 페이지에 머물지 않고, 기관의 공신력과 활동 근거를 축적하는 운영 플랫폼으로 설계한다.

참고 사이트는 기관 소개, 주요사업, 회원사, 정책·제도, 정부사업공고, 자료실, 협업 문의를 하나의 사이트에서 제공하고 있다. 이 구조는 기관 홈페이지의 기본 골격으로 유용하지만, 미래스마트산업진흥원은 협회가 아니므로 회원사 모집 중심 구조를 그대로 적용하지 않는다. 대신 사업 참여, 교육 신청, 전문가 등록, 협력 제안, 연구·정책자료 제공을 중심으로 재구성한다.

## 2. 구축 목적

### 2.1 기관의 정체성과 공신력 확립

① 설립 목적, 비전, 핵심가치, 조직, 주요사업을 일관된 메시지로 전달한다.  
② 기관의 공식 영문명과 약칭인 FSIPI를 통일해 사용한다.  
③ 협약, 교육, 연구, 사업 수행 결과를 지속적으로 공개해 기관의 활동 근거를 축적한다.

### 2.2 주요 사업과 전문역량 소개

① AI·디지털 전환 교육  
② 스마트산업 및 지역혁신 사업  
③ 기업·창업 지원과 컨설팅  
④ 산학연 협력과 전문가 네트워크  
⑤ 정책·연구 및 조사·분석  
⑥ 세미나, 포럼, 전시, 프로젝트 운영

각 사업은 단순 설명보다 대상, 지원 내용, 진행 절차, 기대 효과, 실적과 사례를 중심으로 보여준다.

### 2.3 사업 참여와 문의 전환

방문자가 정보를 읽는 데서 끝나지 않고 다음 행동으로 이어지도록 한다.

- 교육과정 신청
- 행사 참가 신청
- 사업 협력 제안
- 기업 지원 문의
- 전문가·강사 등록
- 자료 다운로드
- 뉴스레터 또는 공지 구독

### 2.4 정보 제공과 콘텐츠 자산 축적

정책·산업 동향, 정부지원사업, 연구보고서, 교육자료, 진흥원 소식을 체계적으로 분류하고 검색할 수 있게 한다. 초기에는 운영자가 선별한 콘텐츠를 제공하고, 향후 자동 수집이나 AI 요약 기능은 출처 검증과 저작권 검토 후 도입한다.

### 2.5 바이브코딩에 적합한 운영 기반 마련

① 화면과 기능을 작은 단위로 정의한다.  
② 콘텐츠와 화면 코드를 분리해 수정 부담을 줄인다.  
③ GitHub를 중심으로 버전 관리와 배포 이력을 남긴다.  
④ 프런트엔드, 백엔드, 데이터베이스를 분리해 단계적으로 확장한다.  
⑤ 관리자 기능을 우선 구현해 개발자 도움 없이 게시물을 운영할 수 있게 한다.

## 3. 핵심 성공 기준

| 구분 | 성공 기준 |
|---|---|
| 정보 전달 | 방문자가 3번 이내의 클릭으로 원하는 사업·자료·문의 화면에 도달 |
| 신뢰성 | 기관 소개, 연락처, 개인정보 처리방침, 주요 실적, 협력기관 정보 제공 |
| 전환 | 교육 신청, 협력 제안, 문의 접수가 정상적으로 저장되고 담당자에게 전달 |
| 운영성 | 관리자가 공지, 사업, 자료, 교육과정, 배너를 직접 등록·수정·삭제 |
| 접근성 | 모바일·태블릿·PC 반응형, 키보드 이동, 대체 텍스트, 명확한 대비 적용 |
| 검색성 | 제목·본문·분류 기반 통합 검색과 게시판별 검색 제공 |
| 성능 | 첫 화면의 불필요한 대용량 이미지와 스크립트를 줄이고 핵심 콘텐츠 우선 표시 |
| 확장성 | 향후 회원·전문가 DB, 신청 관리, AI 검색, 다국어 기능 추가 가능 |

## 4. 구축 범위

### 4.1 1차 MVP 범위

- 메인 페이지
- 진흥원 소개
- 주요사업
- 교육·행사
- 정책·산업정보
- 자료실
- 공지·소식
- 협력기관
- 협업·문의
- 통합 검색
- 관리자 로그인 및 콘텐츠 관리
- 개인정보 처리방침, 이용약관

### 4.2 2차 확장 범위

- 교육과정 온라인 신청 및 상태 확인
- 전문가·강사 등록
- 기업·기관 파트너 등록
- 뉴스레터 구독
- 사업 실적 대시보드
- 다국어 페이지
- AI 기반 자료 검색·요약
- 외부 공공데이터·사업공고 API 연계

## 5. 제외 범위

초기 구축에서는 다음 항목을 제외한다.

- 참고 사이트의 HTML, 이미지, 문구를 그대로 복제하는 행위
- 서버 소스와 데이터베이스의 무단 복원
- 검증되지 않은 정부사업공고의 대량 자동 수집
- 결제, 쇼핑몰, 복잡한 회원 등급제
- 자체 LMS 전체 기능
- 법적 근거가 없는 자격증 발급 기능

## 6. 기본 운영 원칙

① 기관에서 직접 생산한 콘텐츠와 출처가 명확한 외부 자료를 구분한다.  
② 외부 자료는 제목, 요약, 출처, 원문 링크, 게시일을 명시한다.  
③ 개인정보는 필요한 항목만 수집하고 보유기간을 고지한다.  
④ 게시물과 신청 데이터는 관리자 권한에 따라 접근을 제한한다.  
⑤ 홈페이지의 모든 문구와 연락처는 실제 법인 등록 및 운영 정보 확정 후 최종 반영한다.

## 7. 확정이 필요한 사항

- 법인 설립 상태와 공식 표기
- 대표자 직함과 성명
- 주소, 대표전화, 이메일
- 로고와 브랜드 색상
- 주요사업의 공식 명칭과 우선순위
- 교육 신청을 자체 폼으로 받을지 외부 폼으로 시작할지 여부
- 개인정보 보호 책임자와 보유기간
- 도메인과 공식 이메일 계정
