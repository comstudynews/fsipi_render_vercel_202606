# 콘텐츠 구성안

## 1. 콘텐츠 전략

미래스마트산업진흥원 홈페이지는 외부 정보를 많이 모으는 것보다 기관이 실제로 무엇을 하는지 분명히 보여주는 콘텐츠가 우선이다. 초기 콘텐츠는 `기관 정체성`, `사업`, `교육·행사`, `정책·자료`, `성과`, `참여` 여섯 범주로 구성한다.

## 2. 메인 페이지 콘텐츠

### 2.1 메인 메시지 초안

**제목**  
미래 산업의 변화가 현장의 성과로 이어지도록 연결합니다.

**보조 문구**  
미래스마트산업진흥원은 AI·디지털 전환 교육, 스마트산업 연구, 기업 지원, 지역혁신, 산학연 협력을 추진합니다.

**행동 버튼**  
- 주요사업 보기
- 협력 문의

※ 위 문구는 기관의 공식 미션 확정 후 최종 조정한다.

### 2.2 핵심사업 카드 초안

| 사업 | 설명 |
|---|---|
| AI·디지털 전환 교육 | 재직자, 대학생, 구직자, 공공기관을 위한 실무형 교육과정 |
| 스마트산업 연구·정책 | 산업·기술 동향 조사, 정책 연구, 사업 기획과 성과 분석 |
| 기업·창업 지원 | 기술 도입, 사업 기획, 교육, 컨설팅, 지원사업 연계 |
| 지역혁신·스마트빌리지 | 지역 문제 해결형 디지털 서비스와 인재 양성 사업 |
| 산학연 협력 | 대학, 기업, 연구기관, 공공기관의 공동 프로젝트와 네트워크 |
| 전문가 네트워크 | 교육, 자문, 심사, 연구, 프로젝트 수행 전문가 발굴과 연계 |

### 2.3 메인 최신 콘텐츠

- 공지사항 5건
- 모집중 교육·행사 3~6건
- 정책·산업정보 5건
- 최신 자료 3건
- 활동소식 3건

## 3. 기관 소개 콘텐츠

### 3.1 기관개요

- 기관명과 영문명
- 설립 배경
- 설립 목적
- 주요 역할
- 법인 유형과 등록정보
- 대표 연락처

### 3.2 인사말

- 대표자 핵심 메시지
- 산업과 교육 현장의 변화
- 기관이 해결하려는 문제
- 협력 요청

### 3.3 비전 체계 예시

- 비전: 미래 산업과 사람을 연결하는 실천형 혁신 플랫폼
- 미션: 교육, 연구, 기업 지원, 지역혁신을 연결해 지속 가능한 산업 생태계를 조성
- 핵심가치: 현장성, 신뢰, 협력, 포용, 지속가능성

※ 공식 비전과 핵심가치는 설립 목적 및 정관과 일치하도록 확정해야 한다.

### 3.4 조직과 연혁

- 조직도
- 부서별 역할
- 설립과 등록
- 협약
- 교육·사업 수행
- 보고서 발간
- 주요 행사

## 10. 초기 콘텐츠 준비 목록

### 10.1 필수 문서

- 기관 소개문 1부
- 대표자 인사말 1부
- 비전·미션·핵심가치
- 주요사업 6개 설명
- 조직도
- 오시는길
- 개인정보 처리방침
- 이용약관

### 10.2 게시물

- 공지사항 3건 이상
- 교육·행사 2건 이상
- 활동소식 3건 이상
- 정책·산업정보 5건 이상
- 자료실 3건 이상
- 수행사례 2건 이상

초기 게시물이 부족하면 메뉴를 많이 만들지 말고 분류 기능으로 운영한다.

### 10.3 이미지와 디자인 자산

- 로고 원본 SVG 또는 투명 PNG
- 대표자 사진
- 기관·사업 대표 이미지
- 수행사례 사진
- 협력기관 로고
- 파비콘
- SNS 공유 이미지

모든 이미지에 저작권과 사용 허가 여부를 기록한다.

## 11. 콘텐츠 메타데이터

각 콘텐츠에 다음 필드를 공통으로 둔다.

| 필드 | 설명 |
|---|---|
| title | 제목 |
| slug | URL 식별자 |
| summary | 목록과 검색에 표시할 요약 |
| body | 본문 |
| contentType | 공지, 교육, 사업, 자료 등 |
| category | 세부 분류 |
| tags | 기술·산업·대상 키워드 |
| status | 초안, 예약, 공개, 비공개 |
| publishedAt | 게시일 |
| author | 작성자 |
| sourceName | 외부자료 출처 기관 |
| sourceUrl | 원문 링크 |
| thumbnail | 대표 이미지 |
| attachments | 첨부파일 |
| relatedContent | 관련 콘텐츠 |

## 12. 문체와 편집 기준

- 기관명과 사업명은 공식 표기를 사용한다.
- 한 문단에 하나의 핵심 내용을 담는다.
- 과장된 표현과 검증되지 않은 수치를 사용하지 않는다.
- 외부 기관의 문구를 그대로 복사하지 않는다.
- 제목은 방문자가 내용을 예측할 수 있게 작성한다.
- 첨부파일만 올리지 말고 본문에 핵심 내용을 요약한다.
- 날짜는 `2026.06.20.` 또는 `2026년 6월 20일` 중 하나로 통일한다.
- 영문 약칭 FSIPI는 처음 한 번만 전체 명칭과 함께 표기한다.

## 13. 콘텐츠 운영 주기

| 콘텐츠 | 권장 점검 주기 |
|---|---|
| 기관 기본정보 | 변경 즉시 |
| 교육·행사 | 주 1회 이상 |
| 공지사항 | 수시 |
| 정책·산업정보 | 주 1~2회 선별 |
| 정부지원사업 | 마감일 기준 주 2회 점검 |
| 활동소식 | 행사·사업 종료 후 7일 이내 |
| 자료실 | 월 1회 이상 |
| 개인정보 처리방침 | 법·운영 변경 시 즉시 |
